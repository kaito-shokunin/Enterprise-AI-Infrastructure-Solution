# AI 平台快速上手指南 (User SOP)

## 1. 如何访问 (Access)
*   **官方网址**: `https://kaito-ai.zeabur.app` (示例)
*   **登录密码**: 请向管理员申请 `ACCESS_CODE`。进入页面后，在弹窗中输入即可。

## 2. 模型切换 (Model Selection)
为了兼顾成本与效率，建议遵循以下使用建议：
*   **Claude 3.5 Sonnet**: 推荐用于 **编写代码**、**复杂逻辑推导**、**重构 SOP**。
*   **DeepSeek V3**: 推荐用于 **日常翻译**、**文案润色**、**简单信息检索**（成本极低）。
*   **GPT-4o**: 推荐用于 **视觉分析**、**多模态交互**。

## 3. 职人进阶技巧
*   **创建助手**: 点击左侧“助手”图标，可针对特定任务（如：C++ 代码审计）创建预设提示词。
*   **插件使用**: 点击对话框下方的积木图标，可以开启“联网搜索”功能。

## 4. 注意事项 (Safety)
*   **对话清理**: 建议每个独立话题结束后点击“开启新对话”，以节省 Token 消耗并提高 AI 回复的精准度。
*   **隐私保护**: 严禁将公司核心机密代码或未公开的财务数据原样喂给 AI。
